# Изучение Библии

Веб-платформа для совместного изучения Священного Писания, построенная на Jekyll 4.4.1 + Firebase. Подходит для домашних групп, воскресных школ и индивидуального изучения.

## Возможности

### Для учеников
- **5 курсов, 36 уроков** — Иакова, Ефесянам, Галатам, Филиппийцам, Мудрость Соломона
- **Интерактивные вопросы** — отвечайте на вопросы прямо в уроке, ответы автосохраняются
- **Страница «Мои ответы»** — просмотр и печать всех своих ответов по курсу
- **Отслеживание прогресса** — пройденные уроки отмечаются в списке

### Для администраторов
- **Управление пользователями** — добавление, редактирование, удаление учеников через админку
- **Просмотр прогресса** — ответы каждого ученика по всем урокам
- **Статистика уроков** — сколько учеников прошли каждый урок

### Общее
- **Лендинг** с описанием платформы и карточками курсов
- **Форма записи** — встроенная Яндекс-форма для новых учеников
- **Доступ к урокам** — заголовки и список уроков видны всем, содержимое — только после входа
- **Адаптивный дизайн** — работает на десктопе и мобильных устройствах

## Firebase

Проект использует Firebase для хранения данных и авторизации.

### Коллекции Firestore

| Коллекция | Документ | Поля |
|-----------|----------|------|
| `users` | `{uid}` | `name`, `email`, `role`, `created` |
| `answers` | `{uid}_{lessonSlug}` | `q1`, `q2`, ... , `_uid`, `_lesson`, `_savedAt` |

### Настройка Firebase

1. Создайте проект на [console.firebase.google.com](https://console.firebase.google.com)
2. Создайте веб-приложение и скопируйте конфиг в `assets/js/firebase-app.js`
3. **Authentication** → включите провайдер **Email/Password**
4. **Firestore Database** → создайте базу данных
5. **Firestore → Rules** → вставьте содержимое `firestore.rules`

### Создание первого администратора

1. **Authentication → Add user** — создайте пользователя с email и паролем
2. **Firestore → `users`** — создайте документ с ID = UID пользователя:
   ```
   name: "Администратор"
   email: "admin@example.com"
   role: "admin"
   ```
3. Войдите на сайт — в навигации появится ссылка «Админка»
4. Новых учеников добавляйте через админку (`/admin/`)

## Запуск локально

```bash
bundle install
bundle exec jekyll serve
```

Сайт откроется на `http://localhost:4000`

## Добавление нового курса

### 1. Создайте уроки

Файлы в папке `_lessons/` по шаблону:

```markdown
---
title: "Название урока"
order: 1
reference: "Книга глава:стихи"
slug: "course-1"
course: "course"
previous_lesson: ""
next_lesson: "/lessons/course-2/"
---

## Раздел

Текст урока...

<div class="question-block">
<label for="q1">1. Текст вопроса</label>
<textarea id="q1" data-question="q1" placeholder="Ваш ответ..."></textarea>
</div>
```

### 2. Создайте страницу курса

Файл `lessons/course.html` — список уроков с фильтром `where: "course", "course"`.

### 3. Добавьте карточку на страницы

- `lessons/index.html` — карточка курса
- `index.html` — карточка на лендинге (опционально)

## Технологии

- **Jekyll 4.4.1** — генератор статических сайтов
- **Firebase Auth** — авторизация по email/паролю
- **Firestore** — облачная база данных (пользователи, ответы)
- **Vanilla JS** — без фреймворков
