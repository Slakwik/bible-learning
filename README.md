# Изучение Библии

Веб-платформа для совместного изучения Священного Писания, построенная на Jekyll 4.4.1. Подходит для домашних групп, воскресных школ и индивидуального изучения.

## Возможности

### Для учеников
- **6 уроков по Посланию Иакова** — вопросы для размышления, комментарии, практическое применение
- **Сохранение ответов** — ответы на вопросы автоматически сохраняются в профиле
- **Отслеживание прогресса** — пройденные уроки отмечаются в списке

### Для администраторов
- **Управление пользователями** — добавление, редактирование, удаление учеников
- **Просмотр прогресса** — ответы каждого ученика по всем урокам
- **Статистика уроков** — сколько учеников прошли каждый урок

## Структура проекта

```
├── _config.yml          # Конфигурация Jekyll
├── _layouts/            # Шаблоны страниц (default, lesson, admin)
├── _includes/           # Компоненты (header, footer)
├── _lessons/            # Коллекция уроков (Markdown)
│   ├── james-1.md       # Урок 1. Вера и дела
│   ├── james-2.md       # Урок 2. Вера и духовный рост
│   ├── james-3.md       # Урок 3. Вера и общество
│   ├── james-4.md       # Урок 4. Вера и наши слова
│   ├── james-5.md       # Урок 5. Вера и борьба
│   └── james-6.md       # Урок 6. Вера и практика
├── lessons/index.html   # Список уроков
├── admin/index.html     # Админ-панель
├── login.html           # Страница входа
├── index.html           # Лендинг
└── assets/
    ├── css/style.css    # Стили
    └── js/
        ├── auth.js      # Авторизация (localStorage)
        ├── app.js       # Логика уроков (сохранение ответов)
        └── admin.js     # Логика админ-панели
```

## Запуск

```bash
bundle install
bundle exec jekyll serve
```

Сайт откроется на `http://localhost:4000`

## Вход

| Роль | Логин | Пароль |
|------|-------|--------|
| Администратор | `admin` | `admin123` |

Новых учеников можно добавить через админ-панель (`/admin/`).

## Добавление уроков

Создайте файл в папке `_lessons/` по шаблону:

```markdown
---
title: "Название урока"
order: 7
reference: "Книга глава:стихи"
slug: "lesson-slug"
previous_lesson: "/lessons/previous/"
next_lesson: "/lessons/next/"
---

## Раздел

Текст урока...

<div class="question-block">
<label for="q1">1. Текст вопроса</label>
<textarea id="q1" data-question="q1" placeholder="Ваш ответ..."></textarea>
</div>
```

## Технологии

- **Jekyll 4.4.1** — генератор статических сайтов
- **Vanilla JS** — без фреймворков, чистый JavaScript
- **localStorage** — хранение данных пользователей и ответов на стороне клиента
