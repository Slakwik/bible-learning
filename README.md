# Изучение Библии

Веб-платформа для совместного изучения Священного Писания, построенная на Jekyll 4.4.1 + Firebase. Подходит для домашних групп, воскресных школ и индивидуального изучения.

Учебные материалы предоставлены МРО Санкт-Петербургский Центр «Духовное Возрождение» Евангельских Христиан-Баптистов.

## Возможности

### Для учеников
- **8 курсов, 78 уроков** — Иакова, Ефесянам, Галатам, Филиппийцам, Мудрость Соломона, Жизнь христианина, Второзаконие, Евангелие от Марка
- **Интерактивные вопросы** — отвечайте на вопросы прямо в уроке, ответы автосохраняются
- **Страница «Мои ответы»** — просмотр и печать всех своих ответов по курсу
- **Отслеживание прогресса** — пройденные уроки отмечаются в списке
- **Кнопка «Подать заявку»** — форма записи для новых учеников (видна только неавторизованным)

### Для администраторов
- **Управление пользователями** — добавление, редактирование, удаление учеников через админку
- **Просмотр прогресса** — ответы каждого ученика по всем урокам
- **Статистика уроков** — сколько учеников прошли каждый урок

### Общее
- **Лендинг** с цитатой из Писания, карточками курсов и формой записи
- **Страница «О курсе»** — презентации (PDF и HTML), описание платформы
- **Доступ к урокам** — список уроков виден всем, содержимое — только после входа
- **Адаптивный дизайн** — работает на десктопе и мобильных устройствах

## Firebase

Проект использует Firebase для авторизации и хранения данных.

### Коллекции Firestore

| Коллекция | Документ | Поля |
|-----------|----------|------|
| `users` | `{uid}` | `name`, `email`, `role`, `created` |
| `answers` | `{uid}_{lessonSlug}` | `q1`, `q2`, ... , `_uid`, `_lesson`, `_savedAt` |

### Настройка Firebase

1. Создайте проект на [console.firebase.google.com](https://console.firebase.google.com)
2. Создайте веб-приложение и скопируйте конфиг в `assets/js/firebase-app.js`
3. **Authentication** → включите провайдер **Email/Password**
4. **Firestore Database** → создайте базу данных
5. **Firestore → Rules** → вставьте содержимое `firestore.rules`

### Создание первого администратора

1. **Authentication → Add user** — создайте пользователя с email и паролем
2. **Firestore → `users`** — создайте документ с ID = UID пользователя:
   ```
   name: "Администратор"
   email: "admin@example.com"
   role: "admin"
   ```
3. Войдите на сайт — в навигации появится ссылка «Админка»
4. Новых учеников добавляйте через админку (`/admin/`)

## Запуск локально

```bash
bundle install

```

Сайт откроется на `http://localhost:4000`

## Добавление нового курса

### 1. Создайте уроки

Файлы в папке `_lessons/` по шаблону:

```markdown
---
title: "Название урока"
order: 1
reference: "Книга глава:стихи"
slug: "course-1"
course: "course"
previous_lesson: ""
next_lesson: "/lessons/course-2/"
---

## Раздел

Текст урока...

<div class="question-block">
<label for="q1">1. Текст вопроса</label>
<textarea id="q1" data-question="q1" placeholder="Ваш ответ..."></textarea>
</div>
```

### 2. Создайте страницу курса

Файл `lessons/course.html` — список уроков с фильтром `where: "course", "course"`.

### 3. Добавьте карточку на страницы

- `lessons/index.html` — карточка курса
- `index.html` — карточка на лендинге (опционально)

## Адаптация под свою церковь

Если вы хотите использовать эту платформу в своей церкви, вот что нужно изменить:

### Обязательно

| Что | Где | Что поменять |
|-----|-----|-------------|
| Название церкви и ВН | `_includes/footer.html` | Замените «Церковь Преображение» и номер ВН на свои |
| Конфиг Firebase | `assets/js/firebase-app.js` | Создайте свой проект Firebase и вставьте свой `firebaseConfig` |
| Форма записи | `signup.html` | Замените ссылку на Яндекс-форму на свою или создайте свою форму |
| Фон лендинга | `assets/img/hero-bg.jpeg` | Замените на фото вашей церкви или подходящее изображение |

### По желанию

| Что | Где | Что поменять |
|-----|-----|-------------|
| Название сайта | `_config.yml` | Поле `title` |
| Цветовая схема | `assets/css/style.css` | CSS-переменные в `:root` — `--primary`, `--accent` и др. |
| Цитата на главной | `index.html` | Блок `hero-verse` — текст и ссылка на стих |
| Текст на лендинге | `index.html` | Заголовок, описание, карточки возможностей |
| Страница «О курсе» | `about.html` | Информация о материалах, ссылка на организацию |
| Правила Firestore | `firestore.rules` | При необходимости — настройте правила доступа |
| Презентации | `presentation.html`, `presentation.pdf` | Обновите под свою церковь и курсы |

### Порядок действий

1. **Форкните** этот репозиторий
2. Создайте **свой проект Firebase** и вставьте конфиг
3. Замените **название церкви** в footer и на странице «О курсе»
4. Замените **форму записи** и **фон лендинга**
5. При необходимости скорректируйте **цветовую схему**
6. Разверните сайт (GitHub Pages, Netlify, или свой сервер)
7. Создайте **первого администратора** в Firebase (см. раздел выше)

Уроки и курсы можно использовать как есть — они не привязаны к конкретной церкви.

## Технологии

- **Jekyll 4.4.1** — генератор статических сайтов
- **Firebase Auth** — авторизация по email/паролю
- **Firestore** — облачная база данных (пользователи, ответы)
- **Vanilla JS** — без фреймворков
- **Дизайн** — в стиле [spbchurch.ru](https://spbchurch.ru) (Inter + Playfair Display, тёмно-синяя палитра с золотым акцентом)
